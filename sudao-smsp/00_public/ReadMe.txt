========================================================================
    云之讯公共链接库：00_public
========================================================================

本文件概要介绍组成 public 的每个文件的内容。

该目录下存放01_smsp下所有项目工程所共同使用的函数、库等以及他们所对应的源码。
此目录需要特权才可更改


common    : 存放一些共用没有编译成库文件的源码和头文件(所有人可读)
include   : 存放已经被编译成库的头文件(所有人可读)
libraries : 存放已编译好的库文件(所有人可读)
source    : 存放一些开源项目的源码和头文件(所有人可读)

所有共用库编译完成以后，统一输出到libraries中，并把所需要头文件更新至include目录中。

include包括hiredis、json、opencc、ssl、zlib等公共库部分。

source包括xml等开源库部分

--|--|00_public（云之讯公共链接库目录）

例如：

-------------------------------------------------
